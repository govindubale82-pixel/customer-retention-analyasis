# Customer Retention Analysis

## Project Overview
Customer Retention Analysis is a data analytics project that helps businesses understand customer loyalty, retention trends, and churn behavior. The project focuses on identifying retained and churned customers using data visualization techniques.

## Objectives
- Analyze customer retention rate
- Analyze customer churn rate
- Identify customer trends by region
- Improve customer engagement strategies

## Key Performance Indicators (KPIs)
- Total Customers
- Active Customers
- Churned Customers
- Retention Rate (%)
- Churn Rate (%)

## Tools Used
- Microsoft Excel
- Power BI
- GitHub

## Dataset Fields
- Customer ID
- Customer Name
- Region
- Join Date
- Last Purchase Date
- Status

## Dashboard Features
- Customer Retention Rate Card
- Customer Churn Rate Card
- Customers by Region
- Monthly Retention Trend
- Customer Status Distribution
